# realstate (MERN)

Folder structure:
- frontend (React + Vite)
- backend (Node/Express + MongoDB)

## Run locally

### 1) Backend
```bash
cd backend
cp .env.example .env
npm install
npm run dev
```

### 2) Frontend
```bash
cd ../frontend
cp .env.example .env
npm install
npm run dev
```

Open: http://localhost:5173
